✈️ Telegram Bot — Connected
Your expense_bot.py Telegram bot is fully integrated. Commands available in Telegram:
/start — Begin tracking, pick month
/addcat GymFee — Add a new category
/delcat GymFee — Remove a category
/listcats — See all your categories
Type amount — After picking month + cat, just type the number
💡 Setup: Run pip install python-telegram-bot openpyxl then python expense_bot.py. Expenses from the bot go to expenses.xlsx on your server. This web app stores data locally in your browser.
